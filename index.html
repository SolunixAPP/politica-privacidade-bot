Política de Privacidade – SoluneBot

Última atualização: 23 de Junho de 2025

1. Informações Coletadas

O SoluneBot coleta apenas as informações mínimas necessárias para funcionar corretamente, como IDs de usuários, mensagens de comando, canais e servidores. Nenhum dado sensível (como email, IP ou localização) é armazenado.

2. Uso das Informações

As informações são utilizadas exclusivamente para:

  • Executar comandos enviados por usuários
  • Gerenciar sistemas de ticket, logs e moderação
  • Melhorar a experiência dos membros no servidor

3. Compartilhamento de Dados

O SoluneBot não compartilha, vende ou fornece seus dados a terceiros. Toda informação coletada permanece privada e interna ao funcionamento do bot.

4. Armazenamento e Segurança

Os dados armazenados são protegidos com boas práticas de segurança e usados apenas pelo sistema interno do bot. Qualquer dado pode ser removido mediante solicitação do administrador do servidor.

5. Intents Privilegiadas

Este bot utiliza as seguintes intents do Discord:

  • GUILD_MEMBERS – Para sistemas de boas-vindas, tickets e moderação
  • MESSAGE_CONTENT – Para comandos baseados em texto e automações

Essas informações são utilizadas apenas para o funcionamento das funções previstas.

6. Seus Direitos

Se você é dono de um servidor que utiliza o SoluneBot, pode solicitar a exclusão de dados relacionados ao seu servidor entrando em contato.
